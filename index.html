Smart Delivery

A project made for Proximus Belgium.

At the end of the school year of 2015-2016 we had to make a project. The project was chosen by our school, me and 3 friends started to work on this project for Proximus.

The project was inteded to "show off" the possibilities of a new technology called LoRa.
LoRa is a type of a mobile network, but it uses low energy meaning the sender/product would need little to no maintenance for a long time. The batteries are designed to last more than 10 years. LoRa can be combined with other peripherals to make it smarter.

In our case we wanted to use LoRa to make deliveries smarter. We combined the LoRa chip with a GPS reciever to send us the position. When we had the position we could inform the customers the E.T.A of their package, but also communicate with courier if there were any problems.

POIntly

A project powered by the Yelp database (on Android).

For a course at Howest in 2015-2016 I had to make a mobile app on Android. While the project was free of choice it had some requirements. I had to use at least one of 4 Google API's, these were Places, Maps, Hangouts Messaging, Youtube data. I chose Places and Maps, together with the Yelp database.

Each time the app starts up it checks the current location, when it knows the location it goes to Yelp to gather the nearby POI's. When the app finishes downloading, it will show the POI's using a recyclerview (to boost the loading speed).

At the bottom of the app the user can change it's location, category and sorting method. When the user changes one of these the app will ofcourse reload the data with the new parameters. To help choose a location (and make sure it's correct) I used the Google places API. If the user swipes to the right, it will show all the POI's on a map using the Google Maps API.

The user can view details of one POI by tapping on it. On the info screen he can: call, view their website or add it to his favorites.
